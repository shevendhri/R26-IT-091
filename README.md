# GREENCONSTRUCTAI  
### AI-Powered Sustainable Construction & Engineering Platform

GREENCONSTRUCTAI is an intelligent engineering platform designed to support sustainable and climate-responsive construction planning. The system uses AI-assisted analysis and engineering logic to evaluate building concepts, material suitability, blueprint structures, and certification readiness.

---

# Features

## Building Plan Analyzer
- Analyzes conceptual building plans
- Evaluates spatial feasibility
- Supports intelligent room and zoning analysis

## Material Specification Engine
- Recommends suitable construction materials
- Considers climate conditions and sustainability preferences
- Generates engineering-based material recommendations

## Blueprint Analysis
- Performs conceptual blueprint evaluation
- Assesses spatial organization and layout structure
- Supports AI-assisted architectural analysis

## Green Building Certification Pre-Assessment
- Evaluates sustainability readiness
- Performs preliminary green-building compliance analysis
- Supports eco-friendly construction planning.

## Fire Safety Certification Pre-Assessment
- Performs preliminary fire-safety evaluation
- Assesses conceptual safety considerations
- Supports compliance preparation analysis

# Installation Guide

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/GREENCONSTRUCTAI.git
cd GREENCONSTRUCTAI
```

---

## 2. Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## 3. Install Backend Dependencies

```bash
cd backend
pip install -r requirements.txt
```

---

## 4. Configure Environment Variables

Create a `.env` file inside the backend folder and add:

```env
GEMINI_API_KEY=your_api_key
OPENWEATHER_API_KEY=your_api_key
```

---

## 5. Run the Backend Server

```bash
uvicorn app:app --reload
```

---

## 6. Run the Frontend

```bash
cd frontend
npm run dev
```

---

## 7. Open the Application

```bash
http://localhost:3000
```
